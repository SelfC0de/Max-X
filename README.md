## О проекте

Max-X — неофициальный клиент для MAX/oneme.ru на Kotlin + Jetpack Compose.

- Трафик идёт **только** на `api.oneme.ru` — никаких сторонних хостов
- Удалены: аналитика, ИИ-функции, VPN-детектор, Firebase
- E2E шифрование через Android Keystore — опционально, ключи не покидают устройство
- Спуфинг сессии под официальный клиент

## Функционал

| Раздел | Возможности |
|--------|-------------|
| **Чаты** | Список, папки-фильтры, поиск, статус «печатает» |
| **Сообщения** | Отправка, ответ, редактирование, удаление, реакции |
| **Медиа** | Фото, видео, аудио, файлы |
| **Шифрование** | E2E для личных чатов через Android Keystore |
| **Контакты** | Список, онлайн-статус, поиск |
| **Каналы** | Список, подписка, переход в чат |
| **Настройки** | Приватность, прокси, спуфинг устройства |
